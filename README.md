Website yang saya buat adalah sistem informasi stok pakaian. Website ini digunakan untuk membantu pengguna dalam melihat ringkasan data stok melalui dashboard, melihat profil pengguna, serta menampilkan data inventory pakaian dalam bentuk tabel. Website ini dibuat menggunakan Laravel dengan bahasa pemrograman PHP dan menggunakan Blade sebagai templating engine. Tema yang saya pilih adalah pengelolaan inventory fashion, namun pada sistem ini pengguna hanya dapat melihat data stok, belum dapat melakukan pengelolaan seperti menambah, mengubah, atau menghapus data.
